# 项目概览

## 关于 Peers Touch

Peers Touch 是一个现代化的实时通信解决方案，专为开发者设计。

## 核心特性

- **实时通信**: WebSocket 支持
- **多平台**: 跨平台兼容
- **易于集成**: 简单的 API 设计
- **高性能**: 优化的消息处理
